# Library Management System
Kutubxona boshqaruv tizimi — bu kutubxonadagi kitoblarni boshqarish uchun mo‘ljallangan dastur. Dastur yordamida yangi kitob qo‘shish, barcha kitoblarni ko‘rish, kitobni ijaraga berish va qaytarib olish mumkin. Shuningdek, qaysi kitob band yoki bo‘sh ekanini ham ko‘rsatadi. 

## Asosiy kommandalar
1. Kitob qo'shish
2. Kitob ijaraga bersh
3. Barcha kitoblar
4. Kitob qaytarish
5. Chiqish

## 1.1 Kitob qo'shish
Consoledan 1ni tanlasa unda kitob qoshish  munkin
u uchun kitobga id kiritib , Titlega nomini bering , Authorga mualifni bering shundan so'ng kitob qoshiladi.
